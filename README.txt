# SUMMARY #

Taobaoke based on TaoKe API provides by Taobao(Largest c2c website of China) 
Open Platform(TOP). This module can help you build a taoke website with a few 
configurations.


# REQUIREMENTS #

1. php_curl extension.
2. TopSDK.


# INSTALLATION #

* See INSTALL.txt file.


# CONFIGURATION #

See admin/config/taobaoke
